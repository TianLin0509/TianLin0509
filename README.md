## What I Build

I work across two connected tracks:

- **Research code for communication systems**: beamforming, channel estimation, mmWave MIMO, IRS-assisted systems, and deep learning for wireless communication.
- **AI-native productivity tools**: multi-model workspaces, browser extensions, agent/session workflows, and prompt-to-slide automation.

The common thread is practical engineering: turn papers, workflows, and ideas into tools that can be inspected, reproduced, and improved.
